# linq_cpp
Yet Another Cpp Linq Implementation  
My cpp template toy of linq(c#)

## Support Function
* **where**
* **select**
* **all**
* **any**
* **count**
* **sum**
* **average**
* **contains**
* **empty**
* **first**
* **first_or_default**
* **last**
* **last_or_default**
* **distinct**
* **except_with**
* **intersect_with**
* **concat**
